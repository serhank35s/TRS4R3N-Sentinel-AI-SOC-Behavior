# TRS4R3N-Sentinel-AI-SOC-Behavior
GitHub Description (EN) TRS4R3N Sentinel AI is a high-performance HIDS &amp; SOC engine. It correlates Windows/Sysmon telemetry with the MITRE ATT&amp;CK Framework using heuristic chain analysis.
